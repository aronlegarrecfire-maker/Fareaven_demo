# Fareaven_demo
Fareaven is a project created by a group of students who love video games and wanted to create their own. This is actually one of the first versions, so there might be a few bugs and not a lot of content, but much more will be coming soon.

The objective of the game is to survive as long as you can. The difficulty increases over time as you play. When you reach 1000 points, the difficulty will no longer increase.
